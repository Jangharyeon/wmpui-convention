CSS Guide

CSS(Cascading Style Sheets)코드의 기본 작성 가이드입니다.

인코딩

폰트 이름이 영문이 아닐 때 이를 부라우저에서 바르게 표현하고, HTML에서 불러온 스타일을 제대로 렌더링하려면 반드시 CSS인코딩을 선언해야 한다.

					@charset "UTF-8";

기본 규칙

영문 소문자 사용

모든 속성은 영문 소문자로만 작성한다.

					.class {display:block;overflow:hidden;float:left;}

따옴표

공백이 포함된 폰트명, 한글 폰트명, 문자열 데이터 삽입, filter속성에 작은따옴표(‘’)사용
(background url에는 따옴표를 사용하지 않는다.)
단, 인코딩 선언시는 큰따옴표(“”)사용 

					@charset "utf-8";
					font-family:'돋움';
					content:'example';
					background:url(/convention/images/img.png) no-repeat;

주석표기

  • 주석 기호와 주석 사이에 공백 한칸이 있어야 한다.
  • 종료 주석은 사용하지 않는다.
					/* css 주석 */

줄 바꿈, 들여쓰기

속성, 속성값 사이 줄 바꿈, 들여쓰기는 하지 않는다.
단, 중괄호{}가 중첩되는 경우, css3 브라우저별 분기시 사용가능 

					/* 기본 사용 */
					.class-first {display:block;overflow:hidden;float:left;position:relative;z-index:100;width:100%;height:100%;}
					
					/* 예외 */
					.class-second {overflow:hidden;
						-webkit-transition: all 0.3s ease;
						-moz-transition: all 0.3s ease;
						-ms-transition: all 0.3s ease;
						-o-transition: all 0.3s ease;
						transition: all 0.3s ease;
					}
					@media screen and (min-height:1024px) {
						.class {display:block;overflow:hidden;float:left;}
					}

공백

  • 선택자 : 쉼표로 구분되는 부분은 한칸 공백을 넣는다.
  • 선택자와 중괄호 : 선택자와 중괄호 사이는 한칸 공백을 넣고 중괄호 좌우 공백은 제거한다.
  • 속성간 : 속성간 공백은 제거한다.
					.class-first, .class-second, class-third {display:block;overflow:hidden;}

빈줄

의미있는 객체를 구분하기 위해 1줄씩 빈줄생성은 가능

					/* header */
					#header {position:relative;z-index:30;}
					#header .inner {display:inline-block;}
					
					/* content */
					#content {position:relative;z-index:20;}
					#content .inner {display:inline-block;}
					
					/* footer */
					#footer {position:relative;z-index:10;}

선택자

공통선택자(*)

공통선택자는 페이지 성능을 떨어뜨리므로 사용하지 않는다.

id

id 선택자는 레이아웃에만 사용한다.
참고 : 레이아웃

class

최대한 뎁스는 줄이되 어느영역에서 사용하는지 가늠할 수 있게 작성
다중선택자 .class1.class2는 지양한다.

가상선택자

기본적으로 사용되는 :hover, :active, :focus, :visited는 사용가능
이외는 ie8부터 가능한것 들이 많으므로 사용을 지양하지만 :first-child는 ie7이상 가능하다.
참고 : w3schools css-selector

약식 사용 범위

border와 background는 약식 속성을 우선적으로 사용하며, font약식 속성은 사용하지 않는다.

border

border-width, border-style, border-color 순으로 선언한다.
border 초기 선언 시 약식으로 사용하고, 이 후 부분적인 속성이 변경 될 경우 관련 속성(border-width, border-style, border-color)을 선언한다.

background

background-color, background-image, background-repeat, background-position, background-size, background-attachment, background-origin, background-clip 순으로 선언한다.
background 약식 사용은 border와 동일하다.

한글 폰트 선언

영문 폰트만 선언 시 특정 브라우저에서 폰트를 올바르게 출력하지 못하는 경우가 있으므로 한글폰트 선언 시 한글, 영문 폰트를 모두 선언

					font-family:"돋움"; (X)
					font-family:"돋움",dotum; (O)

z-index

속성 값의 범위는 최소 10, 최고 1000이며, 10단위로 증감(10단위 사이의 변수가 생길 경우 1단위값 지정)

					/* z-index 기본 가이드(선택적 사용 가능) */
					#header {z-index:30;}
					#container {z-index:20;}
					#footer {z-index:10;}

media쿼리

  • 반응형 관련 쿼리는 하위에 위치
  • mobile에서 이미지 사이즈 관련 쿼리는 이미지스타일 바로 하단 위치(사이즈 수정시 용이)
  • print관련 쿼리는 최하위 위치

width / height

뷰포트의 너비와 높이

					/* 뷰포트 너비가 768px 이상, 1024px 이하 */
					@media screen and (min-width:768px) and (max-width:1024px) {...}
					
					/* 뷰포트 너비가 768px 이상, 1024px 이하가 아닐 때 실행 */
					@media not screen and (min-width:768px) and (max-width:1024px) {...}

device-pixel-ratio

픽셀 밀도

					/* 해상도 밀도가 1배율 */
					@media screen and (-webkit-min-device-pixel-ratio:1) {...}

					/* 해상도 밀도가 1.5배율 */
					@media screen and (-webkit-min-device-pixel-ratio:1.5) {...}

device-width / device-height

스크린의 너비와 높이

					/* 스크린 너비가 320px, 높이가 480px */
					@media screen and (device-width:320px) and (device-height:480px) {...}

					/* 너비가 320px 이상, 높이가 480px 이상 */
					@media screen and (min-device-width:320px) and (min-device-height:480px) {...}

orientation

뷰포트의 너비와 높이 비율을 이용하여 세로모드인지 가로모드인지 판단

					/* 세로모드 */
					@media screen and (orientation:portrait) {...}

					/* 가로모드 */
					@media screen and (orientation:landscape) {...}

mobile background-size

레티나의 경우 x2, 안드로이드 계역은 x1.5 비율이 생겨 x1 사이즈로 작업했을 경우 흐릿해보이기 때문에 @media query로 분기 처리

					/*
						단말기 해상도 밀도가 1.5배율인 디바이스
						ico@x1.png의 가로 사이즈 100px
						ico@x2.png의 가로 사이즈 200px 일 경우
					*/
					.ico-comm {background:url(ico@x1.png) no-repeat;}
					@media screen and (-webkit-min-device-pixel-ratio:1.5) {
						.ico-comm {background-image:url(ico@x2.png);background-size:100px auto;}
					}
