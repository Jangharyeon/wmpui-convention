의미에 맞는 Naming

  • id는 화면에서의 고유 기능을 명시
  • class는 요소 기능을 명시

prefix(접두사)

접두사를 의미하는 것으로, 앞부분에 사용하며, 주로 형태를 나타내는 데 사용된다.

분류 prefix 부가설명
타이틀 tit 일반적인 타이틀
영역 section 제목 태그(Heading Tag)를 지닌 영역 구분 (중첩사용 지양)
wrap 일반 영역의 묶음 (중첩사용 지양)
inner wrapper가 존재하며 자식 묶음이 단독으로 필요한 경우
네비게이션 gnb 서비스 전체 네비게이션
lnb 지역 네비게이션
snb 사이드 네비게이션
탭 tab
테이블 tbl
목록 list
메뉴 menu
폼 tf textfield (input type text / textarea)
inp input type radio, checkbox, file 등
opt selectbox
lab label
fld fieldset
버튼 btn
박스 box
아이콘 ico
선 line-방향 실선
line-dot-방향 점선
배경 bg
썸네일 thumb
페이징 paging
배너 bnr
텍스트 txt 일반 텍스트
txt-bar 구분선 텍스트
num
copyright
time
강조 emph
링크 link
순서 fst / mid / lst
팝업 popup
레이어 layer
상세내용 desc
댓글 cmt

subfix(하부기호)

하부 기호로서 subfix는 prefix와 함께 부가셜명 용도로 사용한다.

분류 subfix 부가설명
공용 comm 전역
위치변화 top / mid / bot / left / right
순서변화 fst / lst
그림자 shadow
화살표 arr
카테고리 cate
순위 rank

suffix(접미사)

접미사를 의미하는 것으로, prefix와 함께 부가설명 용도로 사용하며 주로 상태를 나타내는 데 사용된다.

분류 subfix 부가설명
상태변화 -on / -off / -over / -hit / -focus
위치변화 -top / -mid / -bot / -left / -right
순서변화 -fst / -lst
이전 / 다음 -prev / -next
버젼 -v1 / -v2 / ...
